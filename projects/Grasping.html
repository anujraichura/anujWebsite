Vision and Control for Grasping

A robotic manipulation project for grasp planning on deformable transparent objects using RGB-D perception and a 7-DOF Sawyer robot.

Context UC Berkeley Robotics
Role Mechanical Lead
Focus Computer vision, controls, PyTorch

Overview

Transparent and deformable objects are difficult for robots to perceive and manipulate reliably. This project explored grasp planning for clear plastic bags using Intel RealSense RGB-D data.

Highlights

  • Developed computer-vision methods for grasp planning on transparent deformable objects.
  • Iterated through grasper designs and optimized end-effector geometry and materials.
  • Implemented a deep-learning grasp pipeline in PyTorch using parallel CNNs for RGB and depth fusion.
  • Improved grasp robustness with K-means denoising and a workspace PD controller for closed-loop arm motion.

Systems View

The project brought together mechanical design, perception, learned grasp selection, and feedback control around a manipulation problem that breaks many standard assumptions.
